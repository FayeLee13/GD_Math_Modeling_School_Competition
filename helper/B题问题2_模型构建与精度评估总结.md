B题问题2：模型构建与精度评估总结

多模型对比-自适应选择预测模型（MCAS-PM）

| 核心结论 问题2的重点不是盲目追求复杂模型，而是建立统一、可比较、可解释的预测框架：用多类模型在不同核能数据集上训练预测，再用公平指标和综合排序选择最优模型。 |
| --- |

1. 问题目标与模型定位

问题2要求在问题1提取特征的基础上划分训练集与测试集，设计合适模型实现预测，并评价不同数据集上的模型表现。题目还要求探讨数据规模、输出维度等因素对模型能力的影响。

| 项目 | 内容 |
| --- | --- |
| 模型名称 | 多模型对比-自适应选择预测模型 |
| 英文简称 | MCAS-PM，Multi-model Comparison and Adaptive Selection Prediction Model |
| 创新方向 | 多模型融合组合创新 |
| 创新本质 | 将线性、局部相似、非线性集成和梯度提升模型放入统一评价框架，根据数据集表现自适应选择最优模型。 |

2. 数据规模和输出维度的含义

“探讨数据规模、输出维度等对模型能力的影响”是要求我们解释为什么不同数据集的预测难度不同，而不是只报告哪个模型指标最好。

| 因素 | 含义 | 对模型能力的影响 |
| --- | --- | --- |
| 数据规模 | 样本数量多少 | 样本少时模型容易过拟合；样本多时训练更充分，测试结果通常更稳定。 |
| 输出维度 | 一个数据集需要预测几个输出变量 | 输出越多，模型越难同时兼顾所有目标，部分输出可能明显更难预测。 |
| 输入维度 | 输入特征数量多少 | 特征多而样本少时容易冗余和过拟合，需要特征选择。 |
| 非线性程度 | 输入输出关系是否复杂 | 非线性越强，线性模型越容易吃亏，树模型和提升模型通常更有优势。 |

3. 多输出数据集的评价逻辑

多输出数据集不能只看一个总误差。正确做法是先对每个输出分别计算指标，再计算整体平均指标，并用稳定性指标判断模型是否只在部分输出上表现好。

| 问题 | 结论 |
| --- | --- |
| 为什么 MAE/RMSE 有尺度问题 | MAE 和 RMSE 与输出变量单位相同，温度、功率、通量等不同量纲不能直接公平平均。 |
| 为什么 R2 没有单位问题 | R2 = 1 - SSE/SST，分子分母单位相同会抵消，因此是无量纲比例指标。 |
| 为什么直接平均 MAE/RMSE 不公平 | 大尺度输出会主导平均误差，小尺度输出的严重错误可能被掩盖。 |
| 如何解决 | 使用 NMAE = MAE/(ymax-ymin)，NRMSE = RMSE/(ymax-ymin)，再对多个输出求平均。 |

| 指标 | 作用 | 使用位置 |
| --- | --- | --- |
| 平均 R2 | 衡量整体解释能力 | 主评价指标 |
| 平均 NMAE | 衡量归一化后的平均绝对误差 | 主评价指标 |
| 平均 NRMSE | 衡量归一化后的大误差水平 | 主评价指标 |
| R2 标准差 | 衡量各输出拟合表现是否稳定 | 辅助判断指标 |
| NRMSE 标准差 | 衡量各输出误差表现是否稳定 | 辅助判断指标 |

输出间指标标准差的含义：对同一模型在多个输出上的指标值计算普通标准差。标准差越大，说明模型在不同输出上的表现差异越明显。

4. 多输出中的模型与特征差异

多输出任务中，不同输出可能适合不同模型，也可能对应不同筛选特征。这是正常现象，因为各输出背后的物理机制、非线性程度和噪声水平可能不同。

| 问题 | 推荐处理方式 |
| --- | --- |
| 不同输出可能适合不同模型 | 主方案采用数据集级最优模型，即每个数据集选一个整体表现最好的模型；补充分析中说明部分输出由其他模型取得更好结果。 |
| 不同输出可能对应不同筛选特征 | 对各输出分别计算特征得分，再取平均作为总体重要性；同时设置核心特征并集保护规则。 |
| 核心特征并集保护规则 | 若某特征对任一输出属于核心特征，则至少保留为候选特征，不直接剔除。 |

5. 候选预测模型

主线模型建议采用 Ridge、随机森林和梯度提升树。如果时间允许，可以加入 KNN 回归作为补充对照模型。

| 模型 | 模型角色 | 是否建议 | 说明 |
| --- | --- | --- | --- |
| Ridge 回归 | 线性基线 | 强烈建议 | 简单、稳定、好解释，用来衡量线性关系能达到的效果。 |
| KNN 回归 | 局部相似性基线 | 可作为补充 | 原理简单，但对标准化敏感，高维时距离可能失效。 |
| 随机森林 | 非线性集成模型 | 强烈建议 | 能捕捉复杂非线性关系，并可输出特征重要性。 |
| 梯度提升树 | 精度增强模型 | 强烈建议 | 表格数据上通常表现较强，适合作为高精度模型。 |

| KNN 的定位 KNN 可以使用，但不要当主角。它更适合作为“局部相似性假设”下的补充对照模型，并且必须先对输入特征做标准化。 |
| --- |

6. 指标主次与最优模型判定

无论单输出还是多输出，都不能只凭单个指标选模型。本文采用加权综合排序法：先对各模型在每个指标上的表现排序，再计算综合排名分。综合排名分越小，模型越优。

| 任务类型 | 主评价指标 | 辅助指标 |
| --- | --- | --- |
| 单输出数据集 | R2、RMSE | MAE |
| 多输出数据集 | 平均 R2、平均 NRMSE、平均 NMAE | R2 标准差、NRMSE 标准差 |

| 单输出综合排名分 Score = 0.4 * R2排名 + 0.35 * RMSE排名 + 0.25 * MAE排名。R2 越大排名越靠前；RMSE 和 MAE 越小排名越靠前。 |
| --- |

| 多输出综合排名分 Score = 0.4 * 平均R2排名 + 0.35 * 平均NRMSE排名 + 0.25 * 平均NMAE排名。综合排名分越小，说明整体表现越好。 |
| --- |

若多个模型综合排名接近，则优先选择 RMSE 或 NRMSE 更低的模型，因为核能系统属于高风险场景，需要尽量避免大误差；若仍接近，则选择输出间指标标准差更小、解释性更好的模型。

7. 问题2完整建模流程

- 读取问题1处理后的数据，保留全特征集和筛选特征集两个版本。
- 划分训练集与测试集，推荐 80% 训练、20% 测试，并固定随机种子。
- 对需要距离计算或线性建模的模型进行标准化，尤其是 Ridge 和 KNN。
- 分别训练 Ridge、KNN、随机森林和梯度提升树模型。
- 单输出数据集计算 R2、RMSE、MAE；多输出数据集计算平均 R2、平均 NMAE、平均 NRMSE 及稳定性指标。
- 使用加权综合排序法选择每个数据集的最优模型。
- 比较全特征模型与筛选特征模型，验证问题1特征选择策略是否有效。
- 分析数据规模、输出维度、输入维度和非线性程度对模型表现的影响。
8. 可直接写入论文的总结段

本文构建多模型对比-自适应选择预测模型，将 Ridge 回归、KNN 回归、随机森林和梯度提升树纳入统一实验框架。对于单输出数据集，采用 R2、RMSE 和 MAE 评价模型性能；对于多输出数据集，分别计算各输出指标，并进一步使用平均 R2、平均 NMAE、平均 NRMSE 及输出间指标标准差评价模型整体精度与稳定性。模型选择采用加权综合排序法，优先选择整体拟合能力强、归一化误差低且输出表现稳定的模型。通过比较全特征模型与筛选特征模型，进一步验证问题1特征选择策略的有效性，并分析样本规模、输出维度和非线性程度对预测能力的影响。
