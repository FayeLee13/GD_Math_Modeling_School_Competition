B题问题3：模型可解释性分析总结

三层物理一致性解释模型（TPCI-EM）

| 核心结论 问题3不是重新设计预测算法，而是基于问题2训练好的模型，构建“物理预期 - 模型依赖 - 输出响应”的解释闭环，判断模型预测是否可信、是否符合物理直觉。 |

1. 问题目标与模型定位

问题3要求考虑问题2中模型的结构设计和决策过程，结合核能系统特点与机理，进行事前、事中、事后的可解释性分析，分析模型是否能够反映物理上的合理关系。

| 项目 | 内容 |
| --- | --- |
| 模型名称 | 三层物理一致性解释模型 |
| 英文简称 | TPCI-EM，Three-layer Physical Consistency Interpretation Model |
| 创新方向 | 跨领域模型迁移创新 |
| 创新本质 | 将机器学习可解释性方法与核能物理合理性校验结合。 |

2. 问题3是否需要重新建模

问题3不需要重新设计新的预测算法，而是使用问题2训练好的最优模型，提取解释信息、绘制解释图表，并进行物理一致性检验。

| 问题3任务性质 更准确地说，问题3是模型解释与可信度评估，而不是新的预测建模任务。 |
| --- |

3. 三层解释框架

问题3采用“事前解释 -> 事中解释 -> 事后解释 -> 物理一致性判断”的结构。

| 层次 | 核心问题 | 主要方法 | 输出结果 |
| --- | --- | --- | --- |
| 事前解释 | 变量理论上为什么可能影响输出？ | 题面变量含义、基础物理常识、少量文献依据 | 变量物理含义与预期影响表 |
| 事中解释 | 模型实际依赖哪些变量？ | 排列重要性、Ridge 系数、树模型特征重要性 | 模型依赖变量排序图 |
| 事后解释 | 关键变量变化时，预测是否合理变化？ | 敏感性扰动、PDP 部分依赖图 | 输出响应趋势图 |
| 一致性判断 | 模型解释是否符合物理直觉？ | 对比物理预期、模型依赖、输出响应 | 高/中/低一致性评价 |

4. 事前解释：变量物理意义

事前解释不是要求掌握高深核物理，而是基于题面变量含义、基础物理常识和少量文献依据，建立变量与输出之间可能存在关系的物理预期。

| 依据来源 | 用法 |
| --- | --- |
| 题面说明 | 优先使用题目已经给出的变量含义。 |
| 基础物理常识 | 只写基础关系，如热量、温度、压力、流量、裂变、吸收等。 |
| 少量文献或资料 | 用于佐证关键词，不做复杂理论展开。 |

写作时应保持保守，避免绝对化表达，推荐使用“可能影响”“通常与……有关”“从物理意义上看”“在该数据背景下可解释为”等表述。

| 关键提醒 有物理意义不等于在当前数据中最重要。事前解释只说明变量理论上可能相关；变量实际重要性需要由事中解释和问题1综合得分进一步判断。 |
| --- |

5. 事中解释：模型依赖变量分析

事中解释回答“模型预测时主要依赖哪些变量”。推荐主方法是排列重要性，其逻辑是：打乱某个输入变量，如果模型性能明显下降，说明模型很依赖该变量。

| 模型 | 解释方式 |
| --- | --- |
| Ridge 回归 | 标准化回归系数绝对值。 |
| 随机森林 | 排列重要性为主；树模型自带特征重要性可作为补充。 |
| 梯度提升树 | 排列重要性为主；树模型自带特征重要性可作为补充。 |
| KNN | 解释性较弱，主要作为对照模型，不作为解释主力。 |

| 避免重复 特征重要性图属于事中解释，用于找“模型依赖谁”；不要在事后解释中重复使用同一类图。 |
| --- |

6. 事后解释：输出响应与物理一致性

事后解释回答“既然模型依赖这些变量，那么这些变量变化时，模型输出是否合理变化”。推荐主方法是敏感性扰动分析。

| 方法 | 含义 | 建议 |
| --- | --- | --- |
| 敏感性扰动 | 固定其他变量不变，将关键变量增加或减少一定比例，如 ±10%，观察预测输出变化。 | 必做，适合每个数据集。 |
| PDP 部分依赖图 | 固定模型，改变一个输入变量，观察模型平均预测值如何随该变量变化。 | 选做，适合代表性数据集或关键变量。 |
| SHAP 图 | 从样本层面解释各变量对预测的贡献。 | 选做，环境和时间允许时作为加分项。 |

敏感性扰动关注局部变化，例如 ±10%；PDP 图关注变量在整个取值范围内变化时的平均影响趋势。对新手来说，敏感性扰动是主方法，PDP 是补充方法。

7. 物理一致性评价

物理一致性不是严格数学证明，而是定性判断：模型依赖的关键变量和输出响应趋势是否符合题面与基本物理直觉。

| 等级 | 判断标准 |
| --- | --- |
| 高一致性 | 重要特征与题面或物理机理一致，扰动趋势合理。 |
| 中一致性 | 大部分重要特征合理，但部分变量解释不清。 |
| 低一致性 | 模型依赖的变量缺乏物理依据，扰动结果反常。 |

| 判断闭环 事前物理预期 -> 事中模型依赖变量 -> 事后输出响应趋势 -> 物理一致性等级。 |
| --- |

9. 问题3完整流程

- 使用问题2得到的最优模型。
- 整理变量物理含义与预期影响方向。
- 使用排列重要性识别模型依赖变量。
- 对关键变量进行 ±10% 敏感性扰动。
- 选取代表性数据集或关键变量绘制 PDP 图。
- 对比物理预期、模型依赖和输出响应。
- 给出高/中/低物理一致性评价。
- 分析模型可信度、局限性和改进方向。
10. 可直接写入论文的总结段

本文构建三层物理一致性解释模型，从事前、事中和事后三个层面对预测模型进行可解释性分析。事前解释基于题面变量含义、基础物理常识和相关文献，建立输入变量与输出变量之间的物理预期；事中解释采用排列重要性等方法识别模型实际依赖的关键变量；事后解释通过敏感性扰动和部分依赖图分析关键变量变化时预测输出的响应趋势。最后，将物理预期、模型依赖和输出响应进行对比，判断模型是否符合核能系统中的物理合理关系，从而提升预测结果的可信度。
