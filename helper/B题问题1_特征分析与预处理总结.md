B题问题1：特征分析与预处理总结

数据-物理双约束特征筛选模型（DP-FSM）

| 核心结论 问题1的核心不是简单删除变量，而是建立一套“数据统计证据 + 物理机理解释 + 冗余控制 + 后续验证”的特征选择体系。 |
| --- |

1. 问题目标与模型定位

问题1要求分析各输入特征与输出变量之间的关系，识别冗余特征或非线性关系，并提出合理的特征选择或降维策略。

本文采用“先做数据质量诊断，再进行综合特征判断”的路线：先用描述性统计检查数据是否适合建模，再从相关性强度、非线性重要性、物理合理性和冗余程度四个角度评价变量。

| 项目 | 内容 |
| --- | --- |
| 模型名称 | 数据-物理双约束特征筛选模型 |
| 英文简称 | DP-FSM，Data-Physics Feature Screening Model |
| 创新方向 | 跨领域模型迁移创新 |
| 创新本质 | 将统计特征筛选方法与核能物理机理解释结合 |

2. 描述性统计：数据质量诊断

描述性统计不直接参与综合评分，它的作用是做建模前的数据体检，判断数据是否存在异常值、尺度差异、缺失值或无信息变量。

| 检查内容 | 作用 |
| --- | --- |
| 样本数 | 判断数据规模是否足够，影响训练测试划分和模型复杂度。 |
| 均值、标准差 | 判断变量尺度和波动程度，决定是否需要标准化。 |
| 最大值、最小值 | 发现异常值或不合理取值。 |
| 中位数、四分位数 | 判断数据是否偏态，辅助识别离群点。 |
| 缺失值 | 判断是否需要填补或删除。 |
| 标准差是否为0 | 判断变量是否几乎没有信息量。 |

3. 综合特征判断依据

在描述性统计完成数据质量诊断后，进一步构建综合特征判断得分：

| 综合得分公式 S = 0.3C + 0.3N + 0.3P - 0.1R |
| --- |

| 符号 | 含义 | 计算方式 |
| --- | --- | --- |
| C | 相关性强度 | Pearson 与 Spearman 相关系数绝对值的平均。 |
| N | 非线性重要性 | 随机森林特征重要性归一化结果。 |
| P | 物理合理性 | 根据题面、基础物理原理和文献进行 0 / 0.5 / 1 评分。 |
| R | 冗余程度 | 该输入变量与其他输入变量绝对相关系数的平均值或最大值。 |

推荐计算方式：C = (|Pearson| + |Spearman|) / 2；R = mean(|corr(Xi, Xj)|), j != i。

如果希望简化实现，也可以使用规则：若某变量与其他输入变量的最大绝对相关系数超过 0.9，则认为存在较强冗余。

4. 物理合理性评分

物理合理性不是要求证明严格定理，而是判断“某个输入变量影响输出，在核能系统背景下是否说得通”。

| 评分 | 判断标准 |
| --- | --- |
| 1.0 | 题面、基础物理原理或文献明确支持。 |
| 0.5 | 物理上可能相关，但关系不够直接。 |
| 0 | 暂时缺乏合理物理解释，可能只是统计偶然相关。 |

| 数据集 | 主要物理依据 |
| --- | --- |
| chf | 沸腾传热、热工水力。 |
| heat | 傅里叶热传导、能量守恒。 |
| xs | 中子裂变、吸收、散射与增殖平衡。 |
| bwr | 中子物理与热工水力耦合。 |
| htgr / microreactor | 控制鼓角度改变中子通量分布。 |
| fp | 燃料热-力耦合、材料性能。 |
| rea | 反应堆动力学、热反馈。 |

5. 特征选择策略

问题1不仅要计算得分，还要提出明确的特征选择策略。本文采用“综合得分驱动的分级特征选择策略”。

| 特征等级 | 判定标准 | 处理方式 |
| --- | --- | --- |
| 核心特征 | S >= 0.65 | 优先保留，作为主要解释变量。 |
| 候选特征 | 0.45 <= S < 0.65 | 暂时保留，进入后续模型验证。 |
| 低优先级特征 | S < 0.45 | 不优先使用，可作为对照。 |
| 高物理合理性特征 | P = 1 | 即使得分不高，也不直接删除。 |
| 高冗余特征 | |corr| > 0.9 | 保留综合得分更高或物理意义更明确者。 |

为避免误删变量，后续问题2中应比较“全特征模型”和“筛选特征模型”的 MAE、RMSE、R2 等指标。如果筛选特征模型精度接近或优于全特征模型，说明特征选择策略有效。

6. 关于降维策略

B题强调模型可解释性，而 PCA 等降维方法会把原始变量混合成主成分，削弱物理解释。因此本文以特征选择为主，PCA 降维只作为补充对照，不作为主要方法。

| 论文推荐表述 考虑到核能系统输入变量具有明确物理含义，本文不优先采用 PCA 等无监督降维方法，而是采用综合得分驱动的特征选择策略，以保留模型解释性。 |

8. 可直接写入论文的总结段

本文首先通过描述性统计完成数据质量诊断；随后构建综合特征判断得分 S，其中相关性强度由 Pearson 与 Spearman 相关系数共同刻画，非线性重要性由随机森林特征重要性表示，物理合理性由题面机理与文献依据进行规则评分，冗余程度由输入变量间绝对相关系数衡量。根据综合得分将特征划分为核心特征、候选特征和低优先级特征，并在后续建模中验证筛选结果的有效性。
